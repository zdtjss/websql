## 产品介绍

WebSQL 是面向数据库管理与数据分析场景的 AI 原生平台，将人工智能大语言模型与经典数据库管理工具有机融合，为用户提供自然语言驱动的智能化数据库操作体验。系统采用 Go + Vue 3 技术栈构建，编译为单一可执行文件，无需外部运行时依赖，可快速部署于物理服务器、虚拟化平台及容器化环境。

### 产品定位

本产品旨在解决传统数据库管理工具操作门槛高、分析效率低、安全保障弱等核心痛点，适用于以下典型场景：

- **数据查询与分析**：业务人员无需掌握 SQL 语法，以自然语言描述分析目标，系统自动完成 Schema 感知、SQL 生成、跨库关联查询及结果可视化
- **报表自动化生成**：支持一键生成包含图表、数据表格的 Excel、PPT、Word 等多格式分析报告，替代传统"导出数据—手动制图—排版整合"的繁琐流程
- **数据库日常运维**：提供 SQL 编辑器、数据浏览与编辑、ER 图管理、备份恢复、结构对比与数据同步、实时监控等全套运维能力
- **数据安全保障**：内置双智能体安全审核机制、四级 RBAC 权限控制、操作审计追溯及生产环境保护，满足企业级安全合规要求

### 核心功能

1. **AI 智能分析引擎**
   - 基于大语言模型的自然语言理解能力，自动识别用户分析意图
   - 自动感知目标数据库 Schema 结构与表间关系，生成准确、优化的 SQL 查询语句
   - 支持跨数据库实例的关联分析，自动拆分子查询并路由执行、合并结果
   - 内置 ReAct 自纠错机制：SQL 执行异常时自动分析错误原因，调整参数重新生成并重试
   - 支持 AI 驱动的 SQL 性能分析，基于 EXPLAIN 执行计划推理优化建议

2. **智能报告生成**
   - 支持一键生成 Excel、PowerPoint、Word 三种格式的数据分析报告
   - 内置柱状图、饼图、折线图、散点图等多类型可视化图表
   - 基于 Python Skill 引擎扩展专业级报告输出能力，适配复杂排版需求

3. **双智能体安全体系**
   - SQLAgent 主智能体：负责 SQL 生成、执行与结果处理，配备 7 层中间件链（权限审核、危险操作拦截、错误恢复、上下文管理等）
   - PermissionAgent 权限智能体：独立审核每次数据操作，按连接—Schema—表—列四级粒度逐项校验权限
   - 危险操作（DROP、TRUNCATE、无条件 UPDATE/DELETE）触发防篡改审批流，原始参数服务端保存，杜绝前端篡改风险
   - LLM 服务不可用时自动降级为程序化权限检查，保障基本安全底线

4. **经典数据库管理**
   - 基于 CodeMirror 6 的 SQL 编辑器，支持语法高亮、Schema 感知智能补全
   - 可视化数据浏览、筛选、排序、编辑，支持批量操作
   - ER 图逆向工程（由数据库自动生成）与正向工程（由设计图生成 DDL）
   - 选择性备份与 AES 加密恢复
   - Schema 结构对比，自动生成差异 ALTER 语句
   - 数据同步：自动识别源与目标差异，生成同步 SQL，分块处理大数据量场景
   - 跨 Schema 全局搜索，并发扫描表、列、索引、视图及数据内容
   - 内置 QPS/TPS、连接池、缓存命中率、进程列表等实时性能监控

5. **权限与审计**
   - 四级 RBAC 权限模型（连接 → Schema → 表 → 列），遵循"最具体优先"匹配原则
   - 支持密码认证、WebAuthn 生物识别认证、第三方 Token 认证三种方式
   - AI 写操作自动记录审计日志；经典模式 UPDATE/DELETE 操作自动备份至历史表
   - SQL 执行熔断器 + AI 调用熔断器 + IP 级限流 + 登录限流，四层容错保障

## 核心能力

### AI 双智能体架构（基于 Eino ADK）

- **SQLAgent**：ReAct 模式主智能体，配备 9 个工具（查询、写入、导出 Excel/PPT/Word、导入数据等）和 7 层中间件链（权限审核、危险 SQL 拦截、错误恢复、上下文摘要压缩等）
- **PermissionAgent**：独立权限审核智能体，每次 SQL 工具调用前校验用户授权范围，解析 SQL 提取表/字段，按"最具体优先"原则逐表逐字段比对。LLM 不可用时自动降级为程序化权限检查

### 防篡改审批流

危险写操作（DROP/TRUNCATE/无 WHERE 的 UPDATE/DELETE）触发中断机制：原始参数保存至 CheckPointStore（15 分钟过期），前端无法篡改 SQL，用户确认后使用服务端保存的原始参数执行。批量 SQL 逐条展示，可按条选择确认/取消。

### Skill 扩展系统

内置 Python Skill 引擎，将 AI Agent 能力扩展至 Python 生态：专业 PPT 生成（8 种图表类型）、Word 报告生成（含封面/摘要/可视化）、跨库大数据量分析。Python 不可用时自动回退 Go 原生实现，保证基础可用。

### 安全体系

- **认证**：密码（MD5 + 盐值）、WebAuthn 生物识别、第三方 Token 三种登录方式
- **授权**：四级 RBAC（连接 → Schema → 表 → 列），"最具体优先"原则。权限以自然语言注入 AI 系统提示词
- **审计**：所有 AI 写操作自动记入审计表；经典模式 UPDATE/DELETE 自动备份至历史表
- **韧性**：SQL/AI 双熔断器 + IP 级限流 + 登录限流（每 IP 每分钟 10 次）

## 快速开始

```bash
# 编译
git clone <repo-url> && cd websql
go build -o websql .

# 初始化数据库（首次运行）
./websql -sql sqlite3-init.sql

# 启动
./websql -port 8080
```

### Docker

```bash
docker build -t websql .
docker run -d -p 443:443 -v ./data:/app/data websql
```

### 前端开发

```bash
cd web-src && npm install
npm run dev     # 开发服务器
npm run build   # 构建到 static/
```
